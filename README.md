# Vibe Coding Template

Version 1.4

Personal template for rapid app development from scratch using Claude Code.
Tech Agnostic.

## Quick Start

### Create new project

In terminal, move to the parent folder of your future project.

```bash
npx degit JulienRobberechts/vibe-template {my-project-name}
code {my-project-name}
```

### Workflow

1. Edit `specs/pitch.md` - Describe your app idea
2. Open Claude Code terminal (Command Palette)
3. Run `/vibe` - Follow the vibe coding by Claude

Good Luck
