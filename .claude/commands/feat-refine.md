# Feature Refinement

Refine feature spec in `/backlog/*.md`.

**Skills Required:**
- workflow-project: Spec management, iteration format

## Workflow

1. **Read Current Spec**
   - Read `/backlog/*.md`
   - Understand current state

2. **Research Approach**
   - Explore codebase for similar features
   - Research best practices
   - Identify technical constraints

3. **Refine Spec**
   - Add missing details to spec
   - Create iteration checklist
   - Follow workflow-project spec format

4. **Validate**
   - Check completeness
   - Verify technical feasibility

## Output
Updated spec in /backlog/*.md with refined details.
